Câmara dos Deputados aprova projeto que tipifica estelionato por falso advogado

Projeto de lei que cria crime específico para golpes cometidos por pessoas que se passam por advogados usando dados de processos judiciais foi aprovado e segue para o Senado.

Projeto de lei que cria crime específico para golpes cometidos por pessoas que se passam por advogados usando dados de processos judiciais foi aprovado e segue para o Senado. O projeto de lei aprovado na Câmara dos Deputados define como crime o estelionato praticado por quem se faz passar por advogado para obter vantagem financeira, utilizando ilegalmente informações de processos judiciais. A proposta inclui penas de reclusão e multa, com agravantes para casos que envolvam múltiplas vítimas, atuação interestadual ou uso indevido da credencial de advogado.

O texto prevê ainda punições específicas para o uso indevido de credenciais de acesso aos sistemas da Justiça, com aumento da pena caso o agente seja profissional do sistema judiciário ou tenha divulgado dados sensíveis. Também há possibilidade de redução da pena para quem colaborar com as investigações.

Medidas cautelares foram incluídas para permitir o bloqueio imediato de valores e chaves de pagamento vinculadas aos investigados em casos de fraude, com possibilidade de devolução emergencial dos valores transferidos. Além disso, o projeto amplia o rol de entidades autorizadas a propor ações civis públicas para combater essas fraudes.

A proposta visa enfrentar de forma abrangente o golpe do falso advogado, que atinge principalmente pessoas vulneráveis e compromete a confiança no sistema judiciário brasileiro. O projeto agora será analisado pelo Senado, onde poderá ser aprovado para se tornar lei.
